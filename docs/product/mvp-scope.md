# MVP Scope

## Included
- Home dashboard with top signals
- Keyword ranking list
- Keyword detail page
- Watchlist for keywords and related stocks
- Alert rules for strong changes

## Deferred
- Social or shared watchlists
- Paid subscription flows
- Full chatbot assistant
- Cross-market portfolio analytics
- Web admin panel beyond minimal ops needs

## MVP Success Conditions
- Daily review can be completed within a few minutes
- Top keywords feel directionally credible
- Alerts are useful without becoming noise
- The app explains why a keyword is ranked
