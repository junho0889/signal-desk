# UI Principles

## Product Feel
Fast, dense, and evidence-first. The app should feel like a compact research terminal, not a lifestyle finance app.

## Visual Rules
- bright base with strong contrast
- use one primary accent and one alert accent
- keep typography highly legible for numbers and short labels
- use small charts only when they improve comparison

## Information Priority
1. score and movement
2. evidence tags
3. related stocks and sectors
4. recent news and events
5. configuration actions

## Interaction Rules
- minimize taps to reach keyword detail
- keep filters visible on ranking screens
- keep alerts reviewable and dismissible
- avoid decorative motion that slows scanning
