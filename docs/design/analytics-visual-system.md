# Analytics Visual System

## Purpose
Set the direction for a more premium, information-dense, MZ-friendly SignalDesk interface.

## Design Goals
- trustworthy at a glance
- dense without feeling cluttered
- modern and sharp rather than generic finance-app chrome
- charts and evidence should feel like primary product surfaces, not decoration

## Required UI Blocks
- ranking movement charts
- trust and freshness badges
- source-mix breakdowns
- evidence timelines
- sector and stock relationship summaries
- risk and contradiction callouts

## Visual Direction
- prioritize strong hierarchy and restrained but intentional color
- distinguish ranking momentum, trust, and risk with separate visual channels
- use charts that can survive small-screen mobile review
- build for users who skim quickly but still want evidence depth

## Collaboration Notes
- design must freeze chart and stat card patterns before broad app implementation
- model and trust lanes must supply explanation-ready fields, not just one aggregate score
