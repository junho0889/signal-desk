# Mobile Next-Phase Plan

## Purpose
Guide the mobile lane as the platform evolves from MVP read models to a richer analytics product.

## Upcoming Mobile Responsibilities
- integrate chart-ready ranking and detail surfaces
- display trust and freshness more prominently
- support multilingual UI surfaces
- preserve fast review flows for ranking, detail, watchlist, and alerts

## Dependencies
- design lane must define chart and card patterns
- storage and model lanes must publish stable explanation fields
- trust lane must define which warnings surface to users directly

## Non-Goals For This Planning Note
- final production visual polish
- personalization and account features
- offline mode
