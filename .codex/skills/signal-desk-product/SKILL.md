# SignalDesk Product

Read `AGENTS.md`, `docs/product/vision.md`, and `docs/product/mvp-scope.md` first.

## Operate
- Keep scope narrow and first-release focused.
- Write product decisions in terms of user value and review speed.
- Define explicit non-goals when rejecting feature sprawl.
- Express features in a way that data, backend, design, and mobile specialists can act on.
