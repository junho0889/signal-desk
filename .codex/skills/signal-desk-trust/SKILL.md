# SignalDesk Trust

Read `AGENTS.md`, `docs/trust/trust-framework.md`, and `docs/data/source-catalog.md` first.

## Operate
- Prefer inspectable trust dimensions over opaque yes-or-no verdicts.
- Keep contradiction, stale data, and low-confidence mapping visible.
- Make trust outputs stable enough for storage and app consumption.
- Avoid claiming certainty where evidence is mixed.
