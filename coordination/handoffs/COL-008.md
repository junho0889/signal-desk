  - `excerpt_text`
  - `published_at`

  - missing evidence -> `metadata_incomplete` / `missing_evidence_fields`
  - malformed outbound link -> `quarantined` / `malformed_outbound_links`
  - weak evidence text/link support -> `accepted_degraded` / `weak_evidence`
  - duplicate replay -> `duplicate` / `duplicate_payload`
